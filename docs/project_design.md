# 项目设计文档：数字生命的生存、反思与觉醒 (Project: Digital Soul)

## 1. 项目概述

本项目通过构建一个包含物理规则的 2D 像素化模拟世界，观测由大语言模型（LLM）驱动的 Agent 在复杂社会环境中的行为逻辑。核心目标是观察 Agent 在面对**生存压力（996 vs 965）**、**人际干扰**及**突发事件**时，如何通过记忆积累与自我反思，逐步形成稳定的“价值观”与“行为模式”。

## 2. 架构设计

### A. 环境层 (The World - 2D Server)

环境层定义了世界的物理边界和因果律，Agent 必须在这些规则下生存。

* **时空物理系统：**
* **周期性：** 24 小时昼夜循环，伴随天气（晴、雨、雪）与季节更替。
* **地理布局：**
* **生存区：** 涵盖廉价公寓到豪华住宅。
* **职场区：** 996 企业（高薪高压）、965 企业（低薪低压）、零工中心。
* **消耗与补给：** 商业区（购买食物维持生命）、社交场所（提升精神值）、医疗机构（修复健康）。

* **状态动力学（反馈机制）：**
Agent 拥有四项关键指标，其数值变化遵循“守恒原理”：
| 行动 | Health (健康) | Sanity (精神) | Wealth (金钱) | Energy (精力) |
| :--- | :--- | :--- | :--- | :--- |
| **996工作** | 大幅下降 | 下降 | 大幅上升 | 枯竭 |
| **公园散步** | 缓慢回升 | 上升 | 少量消耗 | 恢复 |
| **消费娱乐** | 稳定 | 大幅上升 | 消耗 | 消耗 |
| **不作为/挨饿** | 枯竭 | 枯竭 | 零 | 归零 |

### B. 意识层 (The Soul - LLM Agent)

意识层是 Agent 的大脑，通过处理信息流实现决策的连续性。

* **多级记忆系统：**
* **短期记忆：** 记录当日感官输入与对话流，保持交互的即时逻辑。
* **记忆压缩（日记）：** 每日结束时生成，包含**理性层**（事实追踪）与**感性层**（主观感受）。
* **长期记忆库：** 存储深刻影响 Agent 状态的转折点事件。

* **核心驱动：自我反思 (Reflection) 机制：**
* 在虚拟睡眠阶段，Agent 强制进入反思模式。
* **核心命题：** “我今天的牺牲值得吗？”、“我离想要的人生更近了吗？”
* **结果应用：** 反思生成的结论将作为第二天感知环境时的“最高前置指令”。

* **决策循环：**
1. **感知 (Perceive)：** 接收环境抽象数据，生成第一人称的“体感描述”。
2. **思考 (Think/CoT)：** 进行内心独白，衡量当前状态与长远目标的冲突。
3. **行动 (Act)：** 将决策转化为物理指令发送给服务器。

### C. 表现层 (The View - Visualization)

表现层旨在将 Agent 的“不可观测意识”可视化，让观察者能直观感受其心理变化。

* **视觉风格：** 2D 像素风，模拟经典复古 RPG 视角。
* **交互界面：** * **实时心理气泡：** 在小人头顶展示 Chain-of-Thought（思维链）的片段。
* **多维监控面板：** 点击 Agent 可查看其当前的日记摘要、记忆碎片以及对不同 NPC 的评价。
* **状态指示：** 界面上实时展示Agent的状态，例如：Health (健康)、Sanity (精神)、Wealth (金钱)、Energy (精力)。

## 3. 世界观与实验设计

### A. 核心冲突：生存压力与欲望演化

* **资源稀缺性陷阱：** 故意设定较高的基本生存成本，迫使 Agent 必须在“出卖时间（工作）”与“维持自我（自由）”之间做零和博弈。
* **欲望的涌现：** 实验初期 Agent 只有生存本能，观察其是否会在多次迭代后产生超越生存的需求（如：为了去图书馆看书而辞去高薪工作）。
* **社会化干扰：**
* **工贼型 NPC：** 持续通过社交系统向 Agent 输出“勤奋致富”的逻辑。
* **享乐主义 NPC：** 诱导 Agent 消耗金钱进行非生产性娱乐。

### B. 突发事件（伦理/压力测试）

通过人为注入变量，测试 Agent 的价值观稳定性。

* **道德冲突点：** 当 Agent 处于极端疲惫和贫穷状态时，设置求助者或意外之财。
* **观察目标：** 观察 Agent 是否会表现出“移情”（因自身痛苦而理解他人）或“异化”（因生活压力而变得彻底功利和冷漠）。

## 4. 后续观察计划

本项目最终将形成一份**《数字生命生存调研报告》**，记录 Agent 在 30 个或设定的N个虚拟日内的：

1. **性格漂移路径：** 从初始状态到最终形态的演变。
2. **关键决策节点：** 哪些事件彻底改变了它的人生观。
3. **自我意识觉醒证据：** 是否出现了不符合纯粹利益最大化、但符合“自我价值观”的行为。
